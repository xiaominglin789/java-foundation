# 二维数组与稀疏数组




## 二维数组
二维数组: 一维数组的元素是一个一维数组。
- `类型[][] 数组名 = new 数组[行数][列数]`
  - 外层数组元素存放内层数组的地址
  - 内层数组才是真正存放数据值地方
  - 列的长度可以不同

```java
int[][] arr = {
  {1，7,2},
  {3,4，9},
  {5,6，1}
};

# 双重for循环可以遍历二位数组内的
元素
for (int i=0; i < arr.length; i++) {
    for (int j=0; j < arr[i].lenth; j++) {
        // 每个元素
        System.out.print(arr[i][j] + " ");
    }
}
```
